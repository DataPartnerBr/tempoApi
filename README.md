#Tempo Telecom Api - Challenge


# Instalação

`git clone https://github.com/DataPartnerBr/tempoApi.git`

**Nota: use o de sua preferência Yarn ou Npm **

Instalar as dependências com o comando:

`$yarn` ou `$npm install`

### Rodar o servidor api
```
npm start
```

### Rodar o testes api
```
npm test
```
